# Character Bible - Tsuginome AI School Manga LP

## 主人公：ミナミ リョウ（28）
- 関西在住のフリーランスWeb/マーケ支援者。性別は中性的に見える女性寄り、短めの黒髪ボブ、前髪あり。
- 服装：ベージュのワークジャケット、白T、ネイビーのパンツ。小さな黒いショルダーバッグ。ノートPCに薄い緑のステッカー。
- 表情：序盤は焦り・寝不足・強がり。中盤は半信半疑。後半は目に光が戻り、落ち着いた自信。
- 役割：読者の分身。「AIは触っているが仕事に変わっていない」状態から、AI実務化の一歩を踏み出す。

## メンター：桐谷 ソウ（38）
- ツギノメ側の実務家メンター。落ち着いた男性。短い黒髪、薄い丸眼鏡、軽い無精髭なしの清潔感。
- 服装：濃紺ジャケット、黒T、グレーのスラックス。タブレットとペンを持つ。
- 表情：煽らず、核心を突く。大げさな笑顔ではなく、現場を知る静かな説得力。
- 役割：相棒ではない。途中から現れる案内役。主人公を引っ張りすぎず、考え方と順番を示す。

## サブ人物：0期受講生たち（数カットのみ）
- 2〜4人の大人。会社員、副業フリーランス、個人事業主。顔は固定しすぎない。
- 服装は白/ネイビー/グレー/ベージュ中心。派手すぎない。
- 役割：コミュニティ感・実務感を出す背景人物。相棒キャラ化しない。

## Visual style
- Full-color Japanese business manga / vertical webtoon hybrid for smartphone LP.
- Right-to-left manga speech bubble flow inside each page, but vertical scroll continuity between pages.
- Adult-readable ad manga: clean line art, subtle screentone, warm office/cafe lighting, navy shadows, LINE-green UI accents.
- No repeated logos. No page numbers. No section labels like “第1話” or “STEP 1” inside images.
- Avoid tourist Osaka motifs: no Osaka Castle, Tsutenkaku, takoyaki, Hanshin colors.
- Page images must feel connected: gutters and background motifs continue; laptop/green UI/road line motif recurring.
