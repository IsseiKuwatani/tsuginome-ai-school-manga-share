# ツギノメAI実務スクール 漫画LP セリフ台本 v3

対象：フリーランス
CTA：LINE登録で「AI実務化スターターキット」
目的：ツギノメの実務リソース候補を育てながら、AI実務スクール/コミュニティの収益化につなげる

---

## v3で直したこと

ユーザー指摘を反映。Claude Code / Codex は「最低限触るもの」ではなく、今後のフリーランスにとって大前提の武器として再設計。

### 反映ポイント
- 対象を「非エンジニア一般」ではなく、既に仕事をしているフリーランスに寄せた
- 主人公の疑いを強くした。すぐ納得しない
- 「AI実務」「設計力」など抽象語を、SNS運用・資料作成・LP改善・提案書など具体例に落とした
- Claude Code / Codex は大前提に変更。「難しそうだけど、触るとアイデアが次々出る」「今触らないフリーランスは置いていかれる」という危機感を入れた
- 成果物は絞りつつ、Claude Code / Codex を使って“作れる範囲が広がる体験”を中心にした
- 価格は「0期限定」の煽り感を弱め、含まれるもの/条件を明確にした
- 案件トライアルは、保証ではないことに加え、有償トライアル想定・業務例・判断基準を明記
- LINE登録特典を、よりフリーランスの実務に直結する名称にした

---

# キャラクター設定

## 主人公：佐伯ミナミ（仮）
- 34歳、フリーランス
- SNS運用、資料作成、LP改善、ディレクション補助を業務委託で受けている
- ChatGPTは日常的に使っている
- ただし、AI活用が単価アップ・提案力・案件化に結びついていない
- 過去に講座/教材で「買ったけど使い切れない」経験があり、AIスクールに警戒心がある
- 関西在住または大阪に出られる距離感

## メンター：桐谷さん（仮）
- 40代、ツギノメ側の実務家
- キラキラ講師ではない
- AIを使ったリサーチ、提案書、LP改善、営業資料、業務フロー設計、検品の現場を知っている
- 強い言葉で煽りすぎず、ただしClaude Code / Codexへの危機感ははっきり伝えるタイプ

## 受講生たち
- 後半に少し登場
- フリーランス、業務委託、個人事業主
- 相棒キャラにはしない

# v3追記：トーンの基準

- 煽りすぎない。ただし危機感は弱めない。
- 「誰でも簡単」ではなく、「今フリーランスでClaude Code / Codexに触っていないなら、かなり危ない」という現実を伝える。
- 対象は、危機感を持って最初のハードルを越えたいフリーランス。
- 危機感がない人、AIチャットだけで十分だと思っている人、Claude Code / Codexの重要性を感じていない人は対象外。
