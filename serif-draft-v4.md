対象：フリーランス / 業務委託 / AIを実務に入れたい個人事業主

目的：ツギノメの実務リソース候補を育てながら、AI実務スクール/コミュニティの収益化につなげる

## v4で直したこと

- Notion / Figma / MakeをAIツール扱いしない。必要なら「周辺ツール」として扱うが、漫画内では前面に出さない。
- Geminiは「使ってるけど、仕事に変えきれていない」文脈で使用。
- メンターとの出会いは、いきなりオフラインではなく「LINE登録 → 無料資料 → オンライン個別相談」。
- 9番の表現は「たたき台」では弱いので、「実際に動くツールとして使える」に変更。
- コマ割りは縦読み漫画として設計。余白、吹き出し、表情、間を作る。
- 相棒キャラは出さない。主人公＋メンター＋一部受講生のみ。

# 画像化する漫画ページ構成

## 01 ファーストビュー / 焦り

主人公：
「ChatGPTもGeminiも使ってる。」
「でも、納品できる形にするところで止まる。」

黒帯：
Claude Code / Codex時代、もう“文章だけAI”では足りない。

CTA：

## 02 フリーランスの不安
SNS表示：
「Claude Codeで制作が変わる」
「Codexで実装スピードが変わる」
「AIに作らせる人が増えている」

主人公：
「名前は知ってる。」
「でも、正直こわい。」
「エンジニアじゃない私に関係あるの？」

モノローグ：

## 03 スクール不信とLINE資料

「誰でも簡単！」
「AI副業で月収◯◯万円！」

主人公：
「こういうのは無理。」
「でも、無料資料だけなら見てもいいかも。」

AI実務化スターターキット
1. Claude Code / Codex 最初の一歩テンプレ

## 05 実際に動くツールになる
メンター：
「覚えるのはコード暗記ではありません。」

メンター：

主人公：
「触ってみたら、今の仕事で使えるアイデアが出てきそう…。」

## 06 4週間でやること
見出し：
Claude Code / Codexを触れる側に回る4週間

Week1 Web：最初の壁を越える / 自分の仕事を分解

Week3 Web：動くミニツールを作る
Week4 大阪：成果物レビュー / 納品品質チェック

注釈：
いきなりオフラインではなく、まずはオンラインで安心して相談・参加。
